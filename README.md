# 中国語 期末対策ドリル

第5〜11課の語彙を、Anki型の間隔反復とピンイン入力で学習する静的Webアプリ。

- 入口: `index.html`
- 履歴: ブラウザの `localStorage`
- 端末間同期: Private GitHub Gist（任意）
- GitHub Pages公開URL: `https://t-takata-ainamic.github.io/chinese-final/`

ローカル確認は、このディレクトリで `python3 -m http.server 8000` を実行する。

同期にはGists読み書き権限だけを持つFine-grained tokenを使う。同じTokenとGist IDを各端末の「同期」に設定する。写真から判読できた語彙を収録しているため、試験前に教科書との最終照合を推奨。
