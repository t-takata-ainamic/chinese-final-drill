# 中国語 期末対策ドリル

第5〜11課の語彙を、Anki型の間隔反復とピンイン入力で学習する静的Webアプリ。

出題順シャッフル、声調数字の自動変換、単語一覧・習熟度編集、端末音声によるリスニングに対応。習熟度は「日本語→簡体字」「簡体字→拼音」「簡体字→日本語」「音声→簡体字」の4技能を独立して管理する。
音声は端末内の普通話・女性系・高品質音声を優先選択し、学習用に速度とピッチを調整する。

学習ダッシュボードでは、定着率、期限到来、苦手技能、今日の回答数、計画ペース、技能別・課別の習熟分布、直近7日の学習量、優先復習単語を学習履歴からリアルタイム表示する。

復習方式はAnki/FSRSを参考にした3日間の期限逆算型。既存の学習履歴を維持したまま計画期限だけを再設定する。技能ごとに記憶安定度・難易度・想起率・成功日数・失敗回数を保持し、定着には3回以上の成功、2日以上にまたがる成功、2日以上の記憶安定度を要求する。

- 入口: `index.html`
- 履歴: ブラウザの `localStorage`
- 端末間同期: Private GitHub Gist（任意）
- GitHub Pages公開URL: `https://t-takata-ainamic.github.io/chinese-final/`

ローカル確認は、このディレクトリで `python3 -m http.server 8000` を実行する。

同期にはGists読み書き権限だけを持つFine-grained tokenを使う。最初の端末でGistを作成して共有リンクをコピーし、別端末で一度開けば設定と履歴を引き継げる。以後は起動時に取得し、学習後に自動保存する。写真から判読できた語彙を収録しているため、試験前に教科書との最終照合を推奨。
